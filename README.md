# test-for-kasanare

指定時間内のネットワーク通信量を差分から計測するスクリプト

## 説明

このプロジェクトは、10秒間のプロセス別ネットワーク通信量を測定し、結果をスタックグラフで可視化します。

**特徴：**
- バックグラウンドで継続的にプロセスのIO統計をスキャン
- 時間ごとの差分通信量（Mb/s）を計測
- トップ5プロセスと「その他」を分類して表示
- 定時サンプリングによる精密な計測

## セットアップと実行

### 1. 必要なライブラリをインストール

```bash
pip install psutil matplotlib
```

### 2. スクリプトを実行

```bash
python3 monitor_network.py
```

約12秒で計測完了後、ネットワークトラフィックのグラフが表示されます。

## カスタマイズ

`monitor_network.py` の設定部分で以下をカスタマイズできます：

```python
duration = 10  # 計測時間（秒）
top_n = 5      # 表示するトップアプリケーション数
```
